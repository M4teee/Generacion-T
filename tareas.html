Clase Tarea Consigna
1 Primer contacto
Abrir

Escribe un código HTML básico que contenga un título, un párrafo y un enlace a otra página web. !No te olvides de llamarlo Hola mundo!
2 Tags
Abrir

Usa todos los tipos de tags vistos en clase para explicar dentro de la misma página cómo se utilizan para crear estructuras ramificadas en un documental HTML.
3 Hipervinculos
Abrir

Crear una página de elige tu propia aventura utilizando varios hipervínculos a documentos HTML, la temática la podes elegir vos.

4 Contenedores
Abrir

Usa los diferentes tags HTML que se pueden utilizar para crear contenedores vacíos. Luego, con el inspector de elementos, observalos detenidamente.
5 CSS
Abrir

Crea una página web básica con un título, un párrafo y una imagen, y aplica estilos CSS para cambiar el color, el tamaño mejorando su estética.
6 Herencia de estilos
Abrir

Usa algunas de las propiedades vistas para experimentar la herencia de estilos con varios elementos. No te olvides de los atributos class e id
7 Pseudo selectores
Abrir

Usa los distintos pseudo selectores para mostrar como los elementos cambian en función de su estado o comportamiento agregándole dinamismo.
8 Google fonts y tablas
Abrir

Crea una tabla con distinta distribución y estilos, dentro de esta, explica un tópico que elijas libremente y cambia la fuente del texto con Google fonts.
9 Bootstrap
Abrir

Proba las distintas opciones que te da Bootstrap, son muchísimas, así que elegí la que más te llame la atención. Una vez logrado, ¡modificalo!
10 Gryd System
Abrir

Usa el Gryd System para lo que más te gusta, cuando hayas exprimido toda tu imaginación, intenta que sea responsive, que sea tan estético desde tu celular como desde la computadora.
11 JavaScript
Abrir

Vincula tu JavaScript con tu HTML, una vez hecho esto, haz un console.log que te salude.
12 Valores
Abrir

Crea una variable y proporcionale un valor, del tipo que quieras. Luego, usa typeof para determinar qué tipo de valor es, intenta saberlo antes de que te lo digan.
13 Coerción de datos
Abrir

Usa parseInt y parseFloat en distintos ejemplos. Luego, usa los template literals para concatenar cadenas de texto y valores.
14 Condicionales
Abrir

Crea un if que tenga dentro operadores lógicos, si las condiciones del if se cumplen habrá una recompensa, si no, un castigo.
15 Loops
Abrir

Crea una variable de iteración, mientras no se cumplan las condiciones, el loop permanecerá activo, ¡cuidado con los loops infinitos!

¡Bienvenido a la sección de Tareas! Aquí encontrarás todas las tareas de tus clases. Para enviar tus trabajos, solo tienes que hacer clic en el nombre de la tarea y serás llevado a un link donde deberas crear una carpeta con tu nombre y apellido, en la cual subiras tu trabajo. A la derecha, encontrarás la consigna detallada para que sepas exactamente qué se espera de ti. A la izquierda, se mostrará la clase en la que se asignó la tarea para que puedas mantenerte organizado.
