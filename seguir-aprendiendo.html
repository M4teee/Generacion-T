Curso de Desarrollo Web Full Stack

Duración: 3 meses
Cantidad de Clases: 36 clases
Cantidad de Inscriptos: 120 estudiantes

Este curso es una inmersión completa en el desarrollo web full stack, lo que significa que aprenderás tanto el aspecto del front-end como el del back-end de las aplicaciones web. En el front-end, cubriremos HTML, CSS y JavaScript, así como el uso de bibliotecas y frameworks populares como Bootstrap y React. En el back-end, nos sumergiremos en Node.js y aprenderás a construir servidores web y API REST. También abordaremos bases de datos NoSQL como MongoDB y la interacción con bases de datos en tiempo real utilizando Socket.io. Al final del curso, serás capaz de crear aplicaciones web completas, desde la interfaz de usuario hasta el manejo de datos en el servidor.

Curso de Python para Principiantes

Duración: 2 meses
Cantidad de Clases: 24 clases
Cantidad de Inscriptos: 80 estudiantes

Este curso está diseñado para aquellos que deseen adentrarse en el mundo de la programación utilizando Python. Comenzaremos desde cero, cubriendo los conceptos fundamentales de programación como variables, estructuras de control, bucles y funciones. A medida que avances, exploraremos temas más avanzados como la manipulación de datos, la creación de aplicaciones de línea de comandos y la programación orientada a objetos. Python es un lenguaje versátil y ampliamente utilizado en campos como la ciencia de datos, la automatización y el desarrollo web, por lo que este curso te proporcionará una base sólida para futuros proyectos.

Curso de Desarrollo de Aplicaciones Móviles

Duración: 4 meses
Cantidad de Clases: 48 clases
Cantidad de Inscriptos: 150 estudiantes

¿Siempre has soñado con crear tus propias aplicaciones móviles? Este curso te llevará a través de la creación de aplicaciones para dispositivos iOS y Android. Aprenderás a programar en Swift para iOS y en Kotlin para Android, cubriendo temas como la interfaz de usuario, la navegación y la interacción con servicios web. Utilizarás herramientas populares como Xcode y Android Studio para desarrollar y depurar tus aplicaciones. Al final del curso, estarás listo para lanzar tus propias aplicaciones en las tiendas de aplicaciones y construir tu carrera como desarrollador de aplicaciones móviles.

Curso de Desarrollo de Videojuegos

Duración: 5 meses
Cantidad de Clases: 60 clases
Cantidad de Inscriptos: 100 estudiantes

Si eres un apasionado de los videojuegos y sueñas con crear tus propios mundos virtuales, este curso es para ti. Aprenderás a desarrollar videojuegos en 2D y 3D utilizando motores populares como Unity y Unreal Engine.
